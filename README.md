🎮 Emotion-Based AI Game

This project is an AI-powered game that adapts difficulty based on player's emotions.

🚀 Features

- Emotion detection system
- Dynamic difficulty adjustment
- Interactive gameplay using Pygame

🛠️ Tech Stack

- Python
- Pygame
- AI Logic

▶️ How to Run

pip install -r requirements.txt
python game.py

🌟 Future Scope

- Web deployment using pygbag
- Multiplayer mode
- Real-time emotion detection
